• You can create and edit boards, or add a new project to the board here.
  • By clicking on an existing project on the board, you can edit or remove it.
  • There can be more projects on the same board. Each project will then have its own lane.
  • When creating or editing the project, you cannot set a start date in the future.
  • You also cannot set a deadline in the past, or one that is earlier than the start date.
  • If at least one card exist for a certain project, you cannot change that project's start date.
